# Base de Datos y Enlaces de Economía

Este repositorio contiene una colección organizada de enlaces útiles y referencias para el estudio y análisis en diversos campos de la economía, análisis de datos, finanzas y programación.

## Archivos

- `Enlaces_Organizados.md`: Un documento detallado y organizado por categorías (en formato de tablas) que incluye todos los enlaces, junto con una pequeña descripción.
- `Enlaces_raw.md`: El archivo original con los enlaces.

## Categorías Cubiertas

1. Bases de Datos Mundiales y Multilaterales (Banco Mundial, OCDE, FMI, etc.)
2. Bases de Datos Nacionales - EEUU y Perú
3. Finanzas, Bolsa y Mercados
4. Datos Macroeconómicos y Agregadores
5. Buscadores de Papers, Revistas y Documentos (NBER, SSRN, etc.)
6. Herramientas de Programación, Python y Econometría
7. Repositorios de GitHub y Modelos
8. Cursos, Blogs y Recursos de Aprendizaje
9. Datasets para Práctica y Scraping

Si tienes aportes o nuevos recursos, siéntete libre de colaborar o utilizarlos para tus propios análisis.
